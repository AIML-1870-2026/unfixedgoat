How It Works

The Julia set for a complex constant c is defined by iterating z → z² + c for every point z in the complex plane. Points that remain bounded under iteration are in the Julia set (shown dark); points that escape to infinity are colored by how quickly they escape. The boundary between these regions produces infinitely complex fractal structure.
